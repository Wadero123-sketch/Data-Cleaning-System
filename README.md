# Data Cleaning System

A Python-based full-stack application for receiving, parsing, and cleaning large amounts of data with automated cleaning functions.

## Features

✨ **Core Capabilities:**
- 📤 Support for multiple data formats (CSV, JSON, Excel, XML, Parquet, TSV)
- 🧹 Automated data cleaning functions
- 🔍 Duplicate removal
- ❌ Missing value handling (multiple strategies)
- 📊 Outlier detection and removal
- 🔄 Data reformatting and standardization
- 📈 Comprehensive data statistics and validation
- 💾 Support for both SQL and MongoDB databases

🎨 **User Interface:**
- Intuitive web-based dashboard
- Real-time file preview and statistics
- Interactive cleaning configuration
- Detailed cleaning reports

🚀 **Technical:**
- Fast processing of large datasets
- RESTful API for integration
- Docker support for easy deployment
- Scalable architecture

## Technology Stack

- **Backend:** Python, Flask
- **Databases:** PostgreSQL, MongoDB
- **Frontend:** HTML5, CSS3, JavaScript
- **Containerization:** Docker, Docker Compose
- **Data Processing:** Pandas, NumPy, Scikit-learn
